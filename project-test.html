Project Title: Smart Home Hub

Overview

This project demonstrates how to build a custom IoT gateway using the ESP32. We will cover schematic design, sensors integration, and connecting to a mobile dashboard.

Bill of Materials

  • ESP32 Development Board
  • DHT22 Temperature Sensor
  • 0.96" OLED Display (I2C)
  • Custom RTech PCB
